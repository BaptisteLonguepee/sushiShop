# Images Assets

## Welcome Screen Background

Pour l'écran de bienvenue, veuillez placer une image nommée `sushi_background.jpg` dans ce dossier.

Cette image sera affichée en plein écran sur l'écran de bienvenue de l'application.

Format recommandé:
- Format: JPEG ou PNG
- Nom du fichier: `sushi_background.jpg`
- Dimensions recommandées: 1080x1920px (portrait) ou supérieur
- Taille maximale: 2-3 MB

Si aucune image n'est présente, un dégradé de couleur (utilisant `AppColor.primaryColor`) sera affiché en fallback.
